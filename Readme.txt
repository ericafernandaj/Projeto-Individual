Projeto Individual

As cores foi baseada num projeto anterior onde queria por esse fundo, porém naquela ocasião não iria combinar com o propósito.
No header como tínhamos estudado interação com os usuários achei essencial trazer esse conceito para o meu primeiro projeto.
O layout da página criei um container e coloquei as fotos e o parágrafo alinhando onde queria cada elemento.
E o contato criei um formulário onde você pode aumentar sua caixa de texto para enviar-me uma mensagem.
